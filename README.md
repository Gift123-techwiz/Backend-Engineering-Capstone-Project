# VaultPass Backend API

This project is a secure backend API built with Node.js, Express.js, MongoDB, and Mongoose. It was developed as part of a Backend Engineering assignment to demonstrate authentication, authorization, JWT security, account locking, activity logging, and role-based access control.

## Features

* User Registration
* User Login
* Password Hashing with bcrypt
* JWT Authentication
* Role-Based Authorization (User, Moderator, Admin)
* Account Locking After Failed Login Attempts
* Activity Logging
* Admin User Management
* Global Error Handling

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT
* bcryptjs
* dotenv
* Morgan
